// What's This?

This software measures the nominal length and/or 
angle on the image.

/////////////////////////////////////////////////////
// Usage

1. Open the image file.
2. Follow the instruction in the window.
3. If measuring was done, the length or angle will be 
   shown.

[Keyboard Operations]
    Alt+F4                  Close the window
    Ctrl+O                  Open a file
    Alt+Right               Next page
    Alt+Left                Previous page
    Ctrl+G                  Go to the page
    +                       Zoom in
    -                       Zoom out

[Mouse Operations]
    Left Dragging           Input the segment
    Ctrl+Wheel Rotation     Zoom in/out
    Wheel Dragging          Pan the image
    Wheel Rotation          Vertical scroll
    Shift+Wheel Rotation    Horizontal scroll
